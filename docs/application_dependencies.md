### Dependencies

```
- Node v14.15.1 (LTS) or later. While older versions may work, you should keep the button on the latest LTS version

- npm 6.14.8 (LTS) or newer, Yarn may work but not tested for this project

- AWS CLI v2, v1 may work but have not been tested for this project

- EB CLI

- RDS database running Postgres.

- An S3 bucket to store uploaded photos.

```
